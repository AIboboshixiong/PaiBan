# 🚀 赛博排版大师 (Cyber-Gong-ZH)

[![Version](https://img.shields.io/badge/version-v3.1.0_RC-brightgreen.svg)]()

> **"让你的公众号，从此进入赛博未来。"**

**赛博排版大师** 是一款专为 **微信公众号** 创作者打造的高颜值 **Markdown 排版** 工具。它不仅支持将您的文字一键转化为带有 **内联样式 (Inline-CSS)** 的 HTML，更注入了深度的赛博朋克科技美学，让您的文章在信息流中脱颖而出。

---

## ✨ 核心亮点 (Core Highlights)

- ⚡ **飞书级流畅体验**：响应式编辑器，支持 **文字实时解析**。您可以直接从 **飞书文档**、**Notion** 粘贴内容，极致丝滑。
- 📱 **iPhone 智能模拟预览**：
  - 实时模拟 **iPhone 17 Ultra (概念版)** 与 **iPhone 16 Pro Max**。
  - **自适应缩放系统**：预览窗口随布局拉伸自动调整大小，始终保持最佳视角。
- 🎨 **赛博视觉矩阵**：内置多套高阶色彩方案（星空紫、霓虹蓝、矩阵绿等），文字与 UI 颜色同步跃动。
- 🔄 **强悍的撤销重做系统**：自研历史记录栈，完美支持 `Ctrl+Z` 与 `Ctrl+Y`，排版无忧。
- 🖱️ **一键转到公众号**：无需手动复制，点击按钮系统自动同步剪贴板并为您 **跳转微信后台**。
- 😊 **微信定制级表情包**：集成全套微信风格面部表情、数字序号与贴士符号。

---

## 🛠️ 为什么选择赛博排版大师？

传统的 Markdown 转换器往往样式单一，难以适配公众号严格的样式审查。
**赛博排版大师** 采用 **纯内联样式注入逻辑**，确保：
1. **零丢失**：100% 还原预览效果，不被公众号编辑器拦截。
2. **极客感**：独特的代码块高亮与引用块设计，文章质感瞬间提升。
3. **高效率**：集成了常见的 Markdown 语法快捷工具栏，小白也能上手。

## 🚀 快速开始

### 开发环境启动
```bash
# 复制并安装依赖
git clone https://github.com/AIboboshixiong/PaiBan.git
cd PaiBan
pnpm install

# 启动研发引擎
pnpm run dev
```

### 生产环境构建
```bash
pnpm run build
```
