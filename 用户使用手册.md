# CYBER-GONG-ZH 赛博排版大师 | 使用与开发手册

> **版本**: V3.1.0 RC (旗舰发布版)  
> **定位**: 专为微信公众号设计的赛博科技风格 Markdown 转换器。

---

## 📖 第一部分：用户使用手册 (User Guide)

### 1. 核心流程
1.  **输入内容**: 在左侧编辑器中使用标准 Markdown 语法编写文章。
2.  **快捷工具**: 利用顶部工具栏快速执行加粗、引用、代码块等操作。
3.  **智能撤销**: 支持 **撤销 (Ctrl+Z)** 与 **重做 (Ctrl+Y)**，所有工具栏历史操作均可回溯。
4.  **旗舰预览**: 
    *   **iPhone 17 Ultra / 16e**: 自由切换“灵动岛”与“经典刘海屏”外观。
    *   **动态缩放**: 手机模型会随侧边栏拉伸自动调整大小，始终处于最佳显示状态。
5.  **配色矩阵**: 实时切换顶部的彩色方案（推荐使用默认的 **星空紫**）。
6.  **一键出稿**: 点击“一键转到公众号”。

### 2. 交互亮点
*   **表情面板**: 彻底补全的 Emoji 矩阵，包含微信常用脸部表情及工具符号。
*   **字数统计**: 显眼的彩色状态胶囊，实时监控创作进度。
*   **全汉化**: 每一个按钮都有中文提示，极致易用。

## 💻 第二部分：初级程序员指南 (Developer Guide)

### 1. 核心解析逻辑
项目通过 `src/utils/cyber-parser.ts` 进行“即时内联化”。
*   **优势**: 彻底解决公众号过滤 CSS 类名的问题。
*   **代码示例**:
    ```typescript
    // 解析 H1 标签并注入主色调样式
    line = line.replace(/^#\s+(.*)/, (m, g1) => 
      `<h1 style="${theme.h1Style}">${g1}</h1>`
    );
    ```

### 2. 响应式系统 (Responsive Engine)
编辑器与预览区通过 `editorWidth` 状态控制，利用鼠标事件监听器实现自由拖拉。预览区的 `PhoneShell` 使用 `transform: scale()` 配合 `ResizeObserver` 保证在任何宽度下都不溢出且尽可能撑满。

### 3. 环境维护
*   **编辑器字体**: 推荐使用 `Fira Code` 或相似的宽等圆角字体。
*   **底色建议**: `index.css` 中定义了 `#fffef2` 淡雅背景，用于缓解长时间排版的眼部疲劳。
