# CYBER-GONG-ZH 赛博排版大师 | 使用与开发手册

> **版本**: V3.0 (生产力旗舰版)  
> **定位**: 专为微信公众号设计的赛博科技风格 Markdown 转换器。

---

## 📖 第一部分：用户使用手册 (User Guide)

### 1. 核心流程
1.  **输入内容**: 在左侧编辑器中使用标准 Markdown 语法编写文章。
2.  **快捷工具**: 利用顶栏按钮快速插入常用 Markdown 标记，包括 **Emoji 表情面板**。
3.  **智能历史**: 支持 **撤销 (Ctrl+Z)** 与 **重做 (Ctrl+Y)**，点击工具栏按钮或键盘快捷键均可回退误操作。
4.  **旗舰预览**: 
    *   **iPhone 17 Ultra / 16 Pro Max**: 模拟最前沿的灵动岛外观。
    *   **自适应缩放**: 拖动中间滑块改变宽度，手机屏会自动根据空间缩放以展示更多内容。
5.  **一键出稿**: 点击预览区右上角的“一键转到公众号”按钮。
6.  **公众号粘贴**: 系统自动复制并打开微信后台，您只需在后台 `Ctrl + V`。

### 2. 交互亮点
*   **表情包**: 集成微信常用表情、数字序号、箭头、贴士符号。
*   **样式切换**: 顶栏胶囊按钮，文字颜色同步主题，直观显眼。
*   **拖拽布局**: 编辑器与预览区比例由您掌控。
*   **全汉化界面**: 消除语言障碍，专注内容创作。

## 💻 第二部分：初级程序员指南 (Developer Guide)

### 1. 技术栈
*   **框架**: React 18 (TypeScript)
*   **构建工具**: Vite
*   **核心逻辑**: 纯 Inline Style (内联样式) 解析注入
*   **状态管理**: 实现自研轻量级 **Undo/Redo History Stack**

### 2. 项目目录结构
```text
├── src/
│   ├── utils/
│   │   └── cyber-parser.ts   <-- 【解析核心】处理 MD 到内联 HTML 的转换
│   ├── App.tsx               <-- 【主控中心】交互布局与历史记录管理
│   ├── index.css             <-- 全局样式与自定义滚动条
├── 用户使用手册.md                <-- 您当前阅读的文件
└── package.json              <-- 环境依赖
```

### 3. 历史记录系统 (History Engine)
在 `App.tsx` 中通过 `history` 状态数组管理输入的快照。
*   **触发点**: 文本输入变化、工具栏按钮操作、清空操作。
*   **机制**: 当用户操作时，压入新快照；点击撤销时，移动 `historyIndex` 指针并同步 `input` 状态。

### 4. 响应式模拟器 (Adaptive Scaling)
利用 `ResizeObserver` API 实时计算预览区域的父容器宽高。
*   **计算公式**: `scale = Math.min(可用宽度 / 420, 可用高度 / 880)`。
*   **效果**: 手机壳随侧边栏拉伸自动“呼吸”变化大小。
